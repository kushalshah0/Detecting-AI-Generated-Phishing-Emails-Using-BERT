# Postman Test Cases for Phishing Email Detection API

## Base URL
https://your-vercel-url.vercel.app

---

## Test 1: Health Check
**Method:** GET
**URL:** {{baseUrl}}/health
**Expected Response:**
```json
{
  "status": "healthy"
}
```

---

## Test 2: Predict Phishing Email (LSTM)
**Method:** POST
**URL:** {{baseUrl}}/predict
**Headers:**
- Content-Type: application/json

**Body:**
```json
{
  "text": "Your account has been compromised. Click here to reset your password immediately.",
  "model": "lstm"
}
```

**Expected Response:**
```json
{
  "model": "lstm",
  "prediction": "phishing",
  "confidence": 0.90,
  "top_tokens": [
    {"token": "click", "shap_score": 0.15},
    {"token": "compromised", "shap_score": 0.12}
  ]
}
```
